# Single requirements file for the whole project:
#   - megakernel checks (CUDA kernel + parity tests)
#   - inference server (Qwen3-TTS streaming /tts)
#   - Pipecat server (STT -> LLM -> TTS voice agent)
#
# INSTALL ON THE GPU BOX WITH:  bash setup.sh   (recommended — handles ordering)
# or manually:                  pip install -r requirements.txt
#
# CRITICAL — do NOT let pip touch the box's CUDA torch:
#   The Vast CUDA image ships a CUDA-matched torch + torchvision (e.g. 2.11.0+cu130).
#   We deliberately DO NOT pin torch/torchvision here — pinning/upgrading them
#   breaks torchvision's compiled ops (the "torchvision::nms does not exist" error)
#   and forces CPU loads. Use the pre-installed torch. setup.sh installs the rest
#   with --no-deps where needed so nothing drags torch/transformers around.
#
# transformers is pinned to what qwen-tts 0.1.1 requires (4.57.3). Newer
# transformers (5.x) breaks qwen_tts's check_model_inputs decorator.

# --- model + inference server ---
qwen-tts==0.1.1
transformers==4.57.3
soundfile
ninja                # required to JIT-compile the megakernel (csrc/kernel.cu)
fastapi
uvicorn
websockets

# --- Pipecat voice agent (STT -> LLM -> TTS) ---
# (To run ONLY the pipecat server on a laptop without the model/kernel, use the
#  minimal pipecat_server/requirements.txt instead of this whole file.)
pipecat-ai[webrtc,deepgram,silero,runner]==1.3.0
openai
python-dotenv

# NOTE: torch / torchvision intentionally omitted — use the box image's CUDA build.
#       qwen-tts also pulls accelerate/librosa/torchaudio/onnxruntime transitively.
